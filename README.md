## 💐 是什么

基于.Net 5平台的快速开发解决方案，目前系统包含功能有登录，用户管理，应用菜单管理，权限管理，字典管理，退出登录。通过这些基础功能的实现，分享自己对系统框架设计的理解，对ddd设计的理解，希望多多少少对大家学习使用.net core有帮助，也存在不足之处，还望指出。
